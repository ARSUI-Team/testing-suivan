### Before Recording — Checklist
- [ ] `suivan.vercel.app` is live and all pages load cleanly
- [ ] At least 1 testnet pool exists with participants
- [ ] zkLogin works (Google OAuth configured)
- [ ] Transaction endpoint is online (`/api/sponsor`)
- [ ] DeFiLlama API returns real protocol data
- [ ] SuiScan explorer links are valid
- [ ] No console errors, no stuck loading states
- [ ] Browser zoom: 100%. Cursor: visible. No bookmarks bar.

### Production Tips
- Record screen at 1080p 30fps via OBS or QuickTime
- Record voiceover separately in a quiet room
- Edit: screen recording base layer → voiceover sync → B-roll overlay → motion graphics → subtitles
- Keep total runtime under 5:00 — judges stop watching after time limit
- Practice the screen recording flow at least 5 times so the cursor movement is smooth and confident
